# GIT COMMANDS
Git init : initialize Git
Git add <file/full-project> : add a file to the staging area
Git commit -m "<message>" : make a commit
Git status : show file(s) staging areas
Git remote add origin <url> : add remote github repo 
Git remote rm origin <url> : remove remote github repo 
Git log : Show commits
Git merge <branch> : merge branches
Git switch <branch> / Git checkout <branch> : switch to another branch
Git clone <url> : make a copy of a remote repo locally
Git pull <url-alias> <branch> : get remote changes to local
Git push <url-alias> <branch> : push local changes to remote
